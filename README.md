## Getting Started

WHOIS search with Next.js

### Install:

```bash
npm install
# or
yarn install
```

### Run the development server:

```bash
npm run dev
# or
yarn dev
```

### Run the production server:

#### 1. Build

```bash
npm run build
# or
yarn build
```

#### 2. Start

```bash
npm run start
# or
yarn start
```
